# UFSL Bracket Challenge — Testing Guide 🏀

Hey! Thanks for helping us test the bracket app. Here's everything you need to get started.

## The App

👉 **https://ufsl-bracket-test.vercel.app**

## Step 1: Create an Account

- Tap **Sign in with Google** — use any Google account
- ⚠️ You'll see a "Google hasn't verified this app" warning — that's normal for test apps. Click **Continue** to proceed.

## Step 3: Explore!

Here's a walkthrough of what to check out:

1. **Browse the Leaderboard** — see where you rank against everyone else
2. **View "Shawn 2025" bracket** — this is what a solid bracket looks like (walk through the rounds)
3. **View "Surging Bracket"** — this is the current leader. Tap the row to see their picks
4. **View "No Chance" bracket** — this one's hilariously bad. Enjoy.
5. **Send some smack talk** — go to the pool page and drop some trash talk. It's encouraged.

## What We're Looking For

When you're testing, keep an eye out for:

- **Anything confusing** — if you have to think about what to do next, that's a bug
- **Anything broken** — weird errors, things that don't load, buttons that don't work
- **How it feels on your phone** — is it easy to navigate? Can you read everything?
- **Sign-in friction** — was it easy to get in, or did you get stuck?

Just text or DM what you find — screenshots are super helpful!

## Known Limitations

Don't worry about these — they're expected:

- **Google OAuth warning** — the "unverified app" screen is normal for test/dev environments
- **2025 test data** — we're using last year's tournament data for testing. The real 2026 bracket isn't set up yet
- **Some features are still in progress** — if something looks half-done, it probably is
